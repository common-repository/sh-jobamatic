== Description ==

Job-a-matic is a great way to earn additional revenue from your existing website and provide valuable content for your users with a free hosted job board. The Job-a-matic module uses SimplyHired's XML API to display job postings from their database directly within your WordPress site.

To see the full list of features and benefits of Job-a-matic, visit the [Job-a-matic website](http://www.jobamatic.com/).

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to /wp-admin/options-general.php?page=jobamatic (see detailed configuration section below)

== Detailed Configuration Instructions ==

1. Sign up for your [free Job-a-matic job site](https://www.jobamatic.com/a/jbb/partner-register-account)
2. Log in to the Job-a-matic portal with the account you created in step 1 and click on the XML API tab at the top of the page.
3. Scroll down to the "Job Search GET Parameters" section and locate your Publisher ID (pshid) and Job-a-matic Domain (jbd). Enter these values on the plugin settings page in the appropriate fields.
4. Enter the default job search query.
5. Adjust any of the other default search parameters as required.
6. Select an existing page in your WordPress site to attach the search results to or select "Create new page" and the plugin will create a new page to attach the search results to.
7. Adjust any of the other settings to your liking and save the plugin settings.
